# Property Tools

Free real estate tools and calculators on 360Ghar.

## Available Tools

| Tool | URL | Description |
|------|-----|-------------|
| EMI Calculator | `/emi-calculator` | Calculate home loan EMIs |
| Area Converter | `/area-converter` | Convert between Indian area units (sq ft, sq m, marla, kanal, bigha) |
| Area Calculator | `/area-calculator` | Calculate property area |
| Loan Eligibility | `/loan-eligibility-calculator` | Estimate home loan eligibility |
| Capital Gains Tax | `/capital-gains-tax-calculator` | Calculate capital gains tax on property |
| Document Checklist | `/property-document-checklist` | Required documents for property transactions |
| Stamp Duty | `/stamp-duty-calculator` | Calculate stamp duty and registration charges |
| Vastu Checker | `/vastu-checker` | AI-powered Vastu compliance analysis |
| Circle Rates | `/circle-rates` | Government-notified valuation rates |
| Zone Checker | `/zone-checker` | Check property zoning information |

## API Endpoints

These tools are primarily web-based. Some have API equivalents:

- Vastu analysis: `POST https://api.360ghar.com/api/v1/vastu/analyze` (public, 3-min timeout)

## Authentication

Most tools are public. Vastu analysis is also public (no auth required).
