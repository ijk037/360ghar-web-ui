# Locality Intelligence

Access locality data and neighborhood intelligence for Gurugram and Delhi NCR.

## Web Pages

- Browse all localities: `https://360ghar.com/localities`
- Individual locality: `https://360ghar.com/locality/{slug}`

Each locality page includes:
- Connectivity notes and nearby landmarks
- Market signals and price trends
- Neighborhood amenities
- Verification status
- Direct property search links
